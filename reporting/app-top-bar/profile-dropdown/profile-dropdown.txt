- Profile Dropdown Component:
    Enables users to manage their profile settings and view their profile picture.
    Provides access Options (Icons should be present for each option):
    Help
    New features.
    Notification settings.
    Display as member.
    Dark mode (toggle).
    Invite members (conditional).
    Log Out (displayed in red color)

- Dropdown Trigger:
The dropdown appears when the user clicks the profile icon in the top bar.

- Conditional Display:
If the invite members switch is enabled, the Invite Members option will appear in the profile dropdown.
