    Warning Modal Title: You have unsaved changes

    - Description: Click 'Continue Editing' to keep editing or 'Discard Changes' to abandon changes.
    - Continue Editing Button: Continue Editing
    - Discard Changes Button: Discard Changes
    - close Button: Close

    Trigger Condition:
       - When the user clicks the Back button or Create Dashboard button without saving any dashboard changes,
       - And there are unsaved changes,
       - Then the warning modal will appear.
