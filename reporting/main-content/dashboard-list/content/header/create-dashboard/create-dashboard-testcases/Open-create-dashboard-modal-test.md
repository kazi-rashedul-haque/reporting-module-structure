**Title:** Verify that clicking the Create Dashboard button opens the Create Dashboard modal

**Pre-conditions:**
* At least one dashboard exists in the system

**Test Steps:**
1. Login to the workspace as an authenticated user.
2. Navigate to the Dashboard List page.
3. Click on the "Create Dashboard" button.
4. Verify the modal appears on screen.
5. Check all modal elements are present and visible.

• Create Dashboard modal opens successfully

• Dashboard Name textbox appears with proper placeholder

• All buttons are present and functional
• Close button is accessible in top corner
• Modal overlay displays properly
• Form fields initialize empty
• Modal elements are properly positioned
