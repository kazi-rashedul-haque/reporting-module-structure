campaign_dropdown
Options:
View All: Displays all campaigns with their icon.
Select campaign: Allows selection of a specific campaign dashboard.
Filter Behavior: The dashboard updates based on the selected campaign.
Default State: "All campaigns" is selected by default.

search_field: Search
Placeholder: Search
Behavior:
Invalid Input: Only "All campaigns" is displayed if no match is found.
Valid Input: Matching campaigns are displayed.
Select campaign: Selected community shows a checkmark and updates the result.
