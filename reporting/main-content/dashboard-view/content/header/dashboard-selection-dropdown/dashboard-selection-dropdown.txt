Dashboard Selection Dropdown:
Allows users to select a dashboard from the list of available dashboards, including those recently viewed.

Search Field
Placeholder Text: "Search"

Behavior:
Invalid Input: Displays the message "No options" if no matching dashboards are found.
Valid Input: Displays dashboards that match the entered query.
Recent View: A section labeled "Recently Viewed" displays dashboards that were accessed recently.
Dashboard Selection: Clicking a dashboard from the list selects and loads that dashboard.
Selected Dashboard: The name of the currently active dashboard is displayed at the top of the dropdown.

Icon Indicator:
A lightning bolt icon (⚡️) is shown next to system-generated dashboards for visual distinction.
