Campaign Filter
Default Value: "All Campaigns"

On Click:
Displays all available campaigns—both selected and unselected—along with their corresponding icons.

Search Field
Placeholder Text: "Search"

Behavior:
Invalid Input: Displays the message "No options to show" if no matches are found.
Valid Input: Displays campaigns that match the input.

Campaign Grouping:
1. All Campaigns
2. Single Campaigns (Grouped by Communities)

Campaign Selection:
Campaigns can be selected using checkboxes.

Filter Behavior:
The reports displayed on the dashboard update based on the selected campaign(s).
