report_name: link redirect to the report detail page
all the report cards can drag and drop to rearrange the order of reports in the dashboard
Report Usage Count Display:
    If number_of_reports is 0, it shows "0%" in red color.
    If number_of_reports is 1 or more, it shows "100%" in green color.

Filter Condition Display:
    If a report has filters applied, hovering over the report card shows the WHERE condition as a tooltip.

Time Period Display:
   If a report has a selected time period, it is shown on the report card.
   If a dashboard-level filter overrides the report's selected time period.
       Display the message "Dashboard filters are applied" on the report card.
       Show a tooltip on hover with the overridden filter information.

Chart Type & Data Point Count on Hover:
    If the report card contains a Line Chart, Bar Chart, or Dual Axis Chart, it displays the corresponding chart.
    On hover, the number of data points in the chart is shown.
