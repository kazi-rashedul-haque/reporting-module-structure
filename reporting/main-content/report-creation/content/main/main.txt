Chart Configuration (create_chart_image)
Pre-Configuration State: When no data is selected, a placeholder is shown with the message: "Choose your data to start building your report"

Chart Options for Specific Report Types:
1. Line
2. Bar
3. Horizontal Bar
4. Dual Axis

Chart Axis Options:
- X-Axis and Y-Axis configuration panels are dynamically displayed based on the selected chart type.
- Users can define which fields are plotted along each axis.

Chart Context Menu
Available Actions from the Chart Context Menu:
- Download Options: Export the chart in various formats: PNG, JPEG, PDF, SVG
- View in Full Screen: Expands the chart to full screen for enhanced visibility.
- Print Chart: Opens the system print dialog to print the current chart view.
