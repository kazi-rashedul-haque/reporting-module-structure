**Title:** Verify collapse button accessibility compliance

**Pre-conditions:**
* User is logged in as an authenticated user with reporting access

**Test Steps:**
1. Navigate to the "Reporting" application using keyboard only
2. Open an existing report for editing or create a new report
3. Use Tab key to navigate to the collapse button
4. Verify the button receives keyboard focus
5. Press Enter or Space to activate the button
6. Verify the button functionality works via keyboard

**Expected Results:**
* Collapse button is accessible via keyboard navigation
* Tab key can focus on the collapse button
* Button has visible focus indicator when focused
* Enter or Space key activates the button functionality
* Screen reader announces button purpose and state
