Save Filter
Trigger: Clicking "Save Filter" in the Filters Modal opens the "Save Filter" modal.

Save Filter Modal
Modal Title: "Save Filter"

Form Fields:
- Field Label: Filter Name*
- Placeholder: "Filter Name"
- Required: Yes (mandatory field)
- Max Length: 120 characters
- Validation: Must not contain HTML tags (sanitized input)

Action Buttons:
1. Cancel Button:
    - Closes the Save Filter modal without saving.
    - Returns to the Filters Modal with the current configuration retained.

2. Save Button:
    - Saves the current filter configuration under the specified filter name.
    - Returns to the Filters Modal with the filter saved and ready to apply or manage.
